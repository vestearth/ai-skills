## AI Skills

Use `ai-skills/` as reusable engineering guidance.

Recommended routing:
- Bugs/logs/failing tests: `ai-skills/skills/debugging/SKILL.md`
- PR/code review: `ai-skills/skills/code-review/SKILL.md`
- API/proto/mobile contract: `ai-skills/skills/api-contract-review/SKILL.md`
- Go modules/Docker/CI/shared-lib: `ai-skills/skills/dependency-guard/SKILL.md`
- Provider/callback/signature integration: `ai-skills/skills/vendor-integration/SKILL.md`
- Microservice ownership/boundaries: `ai-skills/skills/microservice-boundary-review/SKILL.md`
- Release/deploy/rollback: `ai-skills/skills/release-checklist/SKILL.md`

Actual source code, tests, CI, logs, and production evidence always override AI memory or summaries.
