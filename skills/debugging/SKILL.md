# Debugging Skill

1. Define symptom
2. Collect evidence
3. List candidate causes
4. Eliminate impossible causes
5. Identify root cause
6. Propose minimal fix
7. Verify

Output:
- Symptoms
- Evidence
- Candidate Causes
- Root Cause
- Fix
- Verification
