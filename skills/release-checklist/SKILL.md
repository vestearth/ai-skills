# Release Checklist

## Pre-release

- Build passes
- Tests pass
- Config verified
- Secrets verified
- Migration reviewed

## Rollback

- Rollback plan exists
- Previous version available
- Recovery steps documented

## Monitoring

- Dashboards available
- Alerts configured
- Error tracking verified

## Smoke tests

- Authentication
- Core business flow
- External integrations
- Critical API endpoints

## Output

- Release readiness
- Risks
- Rollback notes
- Verification results
